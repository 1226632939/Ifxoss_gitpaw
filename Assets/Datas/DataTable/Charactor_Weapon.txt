#	武器表
# 依赖关系 角色基本信息表>角色能力表>武器表 
# 三者中的角色的系统名称必须一致
# 本表用以描述武器的效果，具体伤害等非显示内容由角色能力表表达。
# 注意，游戏中不读取此表的武器名词，而是读取角色能力表的武器名称
#	int		string
# 武器类型,名称,穿越角色,穿越墙壁,角色持有武器,发射时武器消失,说明
,1,weapon001,子弹,TRUE,FALSE,TRUE,FALSE,一般的子弹
,2,weapon002,飞行道具,TRUE,FALSE,TRUE,TRUE,投掷出去的武器
,3,weapon003,静止道具,FALSE,FALSE,FALSE,FALSE,放置类武器
,4,weapon004,射线,TRUE,FALSE,TRUE,FALSE,激光枪或类似类型武器
